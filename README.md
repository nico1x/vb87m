| <kbd>Fn</kbd>+<kbd>SUPER</kbd> (3 seconds)         | Swap Left <kbd>SUPER</kbd> and <kbd>Fn</kbd>                     |
| <kbd>Scroll Lock</kbd> (OSx)                       | Brightness -                                                     |
| <kbd>Pause Break</kbd> (OSx)                       | Brightness +                                                     |
